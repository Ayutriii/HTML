Pengalaman saya selama berkuliah di Universitas Negri Makassar prodi Teknik Komputer 2023 kelas Tekom C tentunya sangaaatttt banyakkkk, karena sekarang saya sudah berada di semester ke 3 saya, otomatis saya sudah berkuliah di UNM selama kurang lebih 2 tahun. Selama 2 tahun ini sangat banyak memori indah selama saya berkuliah mulai dari masa PKKMB dimana saya masih malu malu dengan teman teman lainnya karena belum terlalu akrab. Tapi lama kelamaan kami menyatu karena sering bertemu. Nah disinilah masa masa serunya sudah mulai berdatangan, seperti kerja kelompok bersama, pergi makan siang bersama, buka puasa bersama, dann masiih banyak lagi yang tentunya jika saya ceritakan disini text nya akan sangat panjang sekali.

Selain dapat pengalaman bersama teman teman, saya juga mendapatkan banyak ilmu baru yang sangat bermanfaat bagi saya. Salah satu contohnya adalah tugas HTML ini. Selama saya sekolah, saya tidak pernah mengenal yang namanya bahasa komputer apalagi bahasa HTML seperti ini karena saya masuk sekolah menengah atas jurusan IPA bukan sekolah menengah kejurusan. Saya sangat tertarik dengan jurusan saya karena kebanyakan menggunakan bahasa Inggris, yang dimana itu adalah salah satu bidang saya. Dan selama saya berada di dalam saya mendapat banyak pengalaman baru pada bidang komputer ini seperti coding C++, coding Java, coding HTML, dan banyak lagi.
